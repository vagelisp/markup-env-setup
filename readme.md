# Gulp setup for local HTML/CSS

Just a gulp script to take care of all the optimisation and serve the files.

## Features
* SASS
* Purge CSS
* Image optimization
* JS Minification / Mapping
* browserSync ( auto reload )

## Getting Started

 * Pull the poresitory
 * Run `npm install`
 * Run `gulp` 
 
### Prerequisites

* Node
